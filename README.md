**Инструкция для разработчиков:**

Для работы приложения необходимо:
1. Скачать сертификат для АГТУ (astu.org).
2. Добавить его в cacerts (хранилище сертификатов), выбрав необходимый JDK (21).
3. Перезагрузить ПК/JVM.
